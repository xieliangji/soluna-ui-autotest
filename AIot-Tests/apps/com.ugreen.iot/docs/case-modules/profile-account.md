# 个人信息与账号安全用例

本模块记录“个人中心-个人信息”下的头像、昵称、验证码和退出登录相关用例。

## 模块规则

- 当前可执行用例覆盖 iOS 和 Android；两端优先共享业务校验，只在页面结构、弹窗行为或属性断言不同处拆分平台用例。
- 验证码相关用例只发送验证码并校验倒计时，不输入验证码，不执行破坏性变更。
- 退出登录用例会改变登录态，放在已登录 stage 末尾执行。
- 昵称用例会修改账号昵称，必须在 teardown 中恢复原昵称。
- Android 修改密码和注销账号入口直接位于个人信息页；iOS 对应入口按 iOS 页面层级定位。
- Android 验证码页发送按钮进入倒计时前需要短暂等待页面稳定，避免刚进入页面时点击被吞。

### TC020 个人中心-个人信息-用户昵称显示及设置

状态：已实现，iOS/Android 用例。

操作路径：

1. 点击“我的”tab。
2. 点击头像/个人信息入口进入个人信息页。
3. 断言个人信息页可见。
4. 记录原昵称到 case 变量。
5. 首次点击昵称触发 iOS 编辑层。
6. 点击页面顶部区域收起首次昵称编辑键盘。
7. 等待键盘收起。
8. 再次点击页面顶部区域取消首次昵称编辑层。
9. 打开昵称设置弹窗。
10. 输入少于 2 个字符的昵称。
11. 点击确认。
12. 校验昵称长度不足提示。
13. 点击空白区域收起键盘并关闭弹窗。
14. 再次打开昵称设置弹窗。
15. 输入取消场景昵称。
16. 点击空白区域取消设置。
17. 校验昵称未变更。
18. 打开昵称设置弹窗。
19. 输入超过 30 个字符的昵称。
20. 点击确认。
21. 校验昵称按 30 字符保存。
22. 打开昵称设置弹窗。
23. 输入合法昵称。
24. 点击确认。
25. 校验合法昵称设置成功。
26. teardown 中重新打开昵称设置弹窗，恢复原昵称。
27. 校验原昵称恢复成功。

验证点：

- 昵称长度不足时展示提示。
- 点击空白取消后昵称不变。
- 超过 30 字符按 30 字符保存。
- 合法昵称可设置成功。
- 用例结束后恢复原昵称。

注意事项：

- iOS 首次打开 App 后，第一次点昵称时编辑窗口可能掉到系统键盘下面。必须先 prime 一次，再取消编辑，再执行正式断言。
- 昵称过短提示可显示为 `名称需在2-30个字符内`，断言需覆盖等价文案。
- Android 昵称弹窗是底部弹窗，点击上方空白区域可关闭；Android 昵称值断言使用 `text` 属性。
- Android 昵称输入框有 `max-text-length=30`，超过 30 字符会在输入阶段截断，确认后保存 30 字符结果。

Android 操作路径：

1. 点击“我的”tab。
2. 点击头像/个人信息入口进入个人信息页。
3. 断言个人信息页可见。
4. 记录原昵称到 case 变量。
5. 打开昵称设置弹窗。
6. 输入少于 2 个字符的昵称并确认。
7. 校验 `名称需在2-30个字符内` 等长度不足提示。
8. 点击上方空白区域关闭昵称弹窗。
9. 再次打开昵称设置弹窗，输入取消场景昵称。
10. 点击上方空白区域取消设置。
11. 校验昵称未变更。
12. 输入超过 30 个字符的昵称并确认。
13. 校验昵称按 30 字符保存。
14. 输入合法昵称并确认。
15. 校验合法昵称设置成功。
16. teardown 中重新打开昵称设置弹窗，恢复原昵称并校验恢复成功。

### TC021 个人中心-个人信息-修改密码-发送验证码

状态：已实现，iOS/Android 用例。

操作路径：

1. 点击“我的”tab。
2. 点击头像/个人信息入口进入个人信息页。
3. 点击修改密码入口，进入修改密码验证码页。
4. 断言验证码输入框可见。
5. 点击发送验证码。
6. 断言发送验证码按钮进入秒级倒计时。

验证点：

- 修改密码验证码页可进入。
- 发送验证码按钮可触发倒计时。

注意事项：

- 不输入验证码，不修改密码。
- Android 修改密码入口是个人信息页第 5 行。
- Android 验证码容器 id 为 `usiCode`，发送按钮是容器内可点击 `TextView`；发送前等待页面稳定后再点击。

Android 操作路径：

1. 点击“我的”tab。
2. 点击头像/个人信息入口进入个人信息页。
3. 点击“修改密码”入口进入验证码页。
4. 断言验证码输入框可见。
5. 等待发送按钮稳定。
6. 点击发送验证码。
7. 断言发送验证码按钮进入秒级倒计时。

### TC022 个人中心-个人信息-账号注销-发送验证码

状态：已实现，iOS/Android 用例。

操作路径：

1. 点击“我的”tab。
2. 点击头像/个人信息入口进入个人信息页。
3. 点击账号注销入口。
4. 点击中间确认按钮进入账号注销验证码发送页。
5. 断言验证码输入框可见。
6. 点击发送验证码。
7. 断言发送验证码按钮进入秒级倒计时。

验证点：

- 账号注销验证码页可进入。
- 发送验证码按钮可触发倒计时。

注意事项：

- 不输入验证码，不提交账号注销。
- Android 注销账号入口是个人信息页第 6 行。
- Android 中间确认按钮 id 为 `tvCancelAccount`；验证码页容器 id 为 `usiVerificationCode`。
- 发送验证码后只校验倒计时，不点击最终注销按钮。

Android 操作路径：

1. 点击“我的”tab。
2. 点击头像/个人信息入口进入个人信息页。
3. 点击“注销账号”入口进入注销确认页。
4. 点击确认按钮进入验证码发送页。
5. 断言验证码输入框可见。
6. 等待发送按钮稳定。
7. 点击发送验证码。
8. 断言发送验证码按钮进入秒级倒计时。

### TC023 个人中心-个人信息-退出登录

状态：已实现，iOS/Android 用例。

操作路径：

1. 点击“我的”tab。
2. 点击头像/个人信息入口进入个人信息页。
3. 点击退出登录。
4. 在确认弹框中确认退出登录。
5. 断言退出后回到登录页。

验证点：

- 退出登录成功。
- 退出后进入登录页。

注意事项：

- 该用例会改变登录态，放在已登录 stage 末尾。
- Android 退出登录按钮 id 为 `btnLogOut`，确认弹框确认按钮 id 为 `btn_positive`。

Android 操作路径：

1. 点击“我的”tab。
2. 点击头像/个人信息入口进入个人信息页。
3. 点击退出登录。
4. 在确认弹框中确认退出登录。
5. 断言退出后回到登录页。
