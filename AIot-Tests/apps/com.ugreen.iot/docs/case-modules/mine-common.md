# 我的公共功能用例

本模块记录“我的”页下与账号状态无强绑定的公共入口用例。

## 模块规则

- 这类用例通常可以在已登录 stage 和游客 stage 复用。
- 进入子页面后必须返回“我的”页，并用 `common.mineList` 校验回到列表。
- 语言设置用例会改变全局语言，必须放在 stage 最后执行，并在结束前恢复简体中文。

## 用例记录

### TC001 我的-关于

状态：已实现。

操作路径：

1. 点击“我的”tab。
2. 断言“我的”页列表可见。
3. 点击“关于”入口。
4. 断言关于页根容器可见。
5. 通过 source 正则校验 App 图标节点存在。
6. 校验版本号格式。
7. 点击返回。
8. 断言已回到“我的”页。

验证点：

- 关于页可进入。
- App 图标节点存在。
- 版本号符合版本格式。
- 返回后仍在“我的”页。

注意事项：

- iOS 关于页 App 图标节点可能被 Appium 标记为不可见，因此使用 source 正则校验节点存在。

### TC002 我的-语言设置

状态：已实现。

操作路径：

1. 点击“我的”tab。
2. 断言“我的”页列表可见。
3. 点击“语言设置”入口。
4. 断言语言列表可见。
5. 校验第一个语言标题和第二个语言标题。
6. 选择 English 并确认。
7. 断言切换后返回“我的”页。
8. 通过 source 校验“我的”页语言已切换为 English。
9. 再次进入语言设置页。
10. 选择简体中文并确认。
11. 断言恢复语言后返回“我的”页。
12. 通过 source 校验语言已恢复为简体中文。

验证点：

- 语言列表可见且基础语言项存在。
- 可切换到 English。
- 用例结束前恢复简体中文。

注意事项：

- 该用例会改变 App 全局语言，放在当前 stage 最后执行。
- 不要依赖不稳定的右侧当前语言值子树，优先用页面 source 中的语言值校验切换结果。

### TC003 我的-系统权限管理

状态：已实现。

操作路径：

1. 点击“我的”tab。
2. 断言“我的”页列表可见。
3. 断言系统权限管理入口可见。
4. 点击系统权限管理入口。
5. 断言系统权限管理列表可见。
6. 校验蓝牙、地理位置、麦克风、相册、相机权限项存在。
7. 点击返回。
8. 断言已回到“我的”页。

验证点：

- 系统权限管理入口可进入。
- App 内权限管理页展示预期权限项。

注意事项：

- 当前用例只验证 App 内系统权限管理页，不跳转系统设置页。
